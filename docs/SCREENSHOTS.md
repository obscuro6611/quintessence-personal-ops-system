# Screenshots

This folder should contain portfolio-safe screenshots of Quintessence.

Do not include screenshots showing private notes, real applications, real evidence, phone numbers, email addresses, credentials, or sensitive personal data.

## Recommended Screenshots

Capture the following screens using only sample or fictional data:

1. Login screen
2. Mission Control
3. Evidence Repository
4. Capabilities
5. Career
6. Weekly Review
7. Global Search
8. Timeline
9. Backup / Export
10. Settings

## Screenshot Safety Checklist

Before committing screenshots, check:

- no real applications
- no real companies unless intentionally fictional/example
- no personal evidence
- no private notes
- no credentials
- no tokens
- no real weekly review content
- no private file paths if sensitive
