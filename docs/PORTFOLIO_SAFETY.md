# Portfolio Safety Notes

This repository is intended as a public portfolio version of Quintessence.

Quintessence is a local-first personal operations system. The public repository should contain source code, documentation, architecture notes, setup instructions, and sample data only.

## Do not commit

- backend/.env
- frontend/.env
- backend/data/
- backend/storage/
- backend/backups/
- backend/exports/
- SQLite database files
- uploaded evidence files
- real job application data
- real weekly reviews
- real notes
- personal records
- credentials, passwords, API keys, or tokens

## Safe to commit

- source code
- docs
- screenshots with no private data
- sample data
- .env.example files
- README files
- launcher scripts
- architecture notes

## Working rule

The GitHub version is the portfolio version.

The real local working version may later contain private data, but that private data must remain outside Git tracking.
