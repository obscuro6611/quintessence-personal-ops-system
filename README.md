# Quintessence

Quintessence is a local-first personal operations system for managing projects, capabilities, evidence, career opportunities, AI workflow notes, decision logs, and weekly review.

It was built as a private, local-first operating system for execution, evidence management, capability growth, knowledge organisation, and decision traceability.

## Core Modules

- Mission Control
- Projects
- Career
- Capabilities
- FGAs
- Evidence
- AI Workspace
- Knowledge Vault
- Decisions
- Weekly Review
- Global Search
- Timeline
- Backup / Export
- Settings

## Technology Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic

## Local-First Design

Quintessence is designed to run locally on one machine. It does not require cloud storage, external authentication, or external AI services.

Private runtime data such as databases, uploads, backups, exports, and environment files should not be committed to GitHub.

## Portfolio Context

This repository is intended as a portfolio-safe version of Quintessence. It demonstrates full-stack development, local-first architecture, evidence management, workflow design, knowledge organisation, and systems thinking.
